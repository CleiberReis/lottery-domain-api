# Api lottery 
Api domain about lottery brazil

# prerequisite
* node 14.5.x
* npm 6.x
* docker 20.10.x
* docker-compose 1.27.x

# Installation
```bash
$ npm install
```

# Running the app
```bash
docker-compose up -d
npm run start:local
```
